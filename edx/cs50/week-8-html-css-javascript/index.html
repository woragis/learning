Protocols

IP internet protocol

Wires connecting every possibility of network? No

That's why we use routers

each router pass the connection search to another router, until you find what you are searching

Ip gets information from point A to point B

Ip brakes the data into small "packets"

TCP Transfer Control Protocol

It has total control over the packets that are being transferred through the routers

That's why the real protocol is called TCP/IP because the transfer occurs through ip protocol, but with the help of TCP

Other Protocols

  • FTP File transfer protocol uses port 21
  • SMTP email transfer protocol uses port 25
  • DNS uses port 53
  • HTTP Web Browsing uses port 80
  • HTTPS secure web Browsing uses port 443
